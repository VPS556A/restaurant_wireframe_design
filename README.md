# Restaurant Website Wireframe

## Overview

This project contains low-fidelity wireframes for a Restaurant Website created in Figma as part of a UI/UX wireframing task.

## Pages Included

* Home Page
* About Page
* Services Section

## Prototype

**Figma Prototype Link:**
https://www.figma.com/proto/qn4y0Rxi3kZK1ERCPxs7xX/Restaurant-Website-Wireframes?node-id=0-1&t=twt2aiwSwZfgkvcR-1

## Important

For the best viewing experience:

1. Open the Figma Prototype link.
2. Click the **Display Settings** icon in the top-right corner.
3. Select **Fit Width** under **Scaling Options**.

This ensures the prototype is displayed correctly across different screen sizes.

## Objective

The wireframes focus on:

* Layout Structure
* Content Placement
* Navigation Flow
* User Experience Planning
